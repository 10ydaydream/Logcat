#### 集成步骤

    dependencies {
        debugImplementation 'com.hjq:logcat:3.0'

#### 框架亮点

* 支持日志捕捉开关功能

* 支持日志内容拼接功能

* 支持日志内容收缩展开

* 支持日志复制分享功能

* 支持日志搜索文本高亮
